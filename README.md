# 📊 Smart Stock Portfolio Optimizer

A web-based financial analysis tool that helps users analyze and optimize their stock portfolio using risk metrics, expected return, and portfolio allocation strategies.

This project helps investors understand portfolio performance and visualize asset allocation using interactive charts.

---

## 🚀 Features

- Add multiple stocks to a portfolio  
- Calculate expected return  
- Analyze portfolio risk (volatility)  
- Portfolio allocation visualization using charts  
- Clean and responsive UI  

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Chart.js
